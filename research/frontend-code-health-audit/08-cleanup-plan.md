## Exit criteria

- P0 contract tests pass on both layers; no known role or payload drift remains.
- Alias-aware architecture checks are blocking only after documented SCC debt reaches zero.
- Knip reports no unexplained unused exports/members, and the complete file inventory has no unresolved deletion candidate.
- Page/controller changes preserve URL state, independent loading/error sections, and domain-owned API behavior.
- Browser interaction coverage exists for the custom select, binder receive, dashboard workflows, and permission matrices before their high-risk migrations land.
